Bogl Explorer

* This is a prototype tool, and a work in progress...

This tool is meant to help structure your understanding of BoGL programs. If you are learning BoGL, this tool can help you understand what BoGL language features you need to cover to go from one known BoGL program to a new program.

  1. Enter a Known BoGL program in the top box.
  2. (optional) Enter a Goal BoGL program you want to know in the bottom box (ideally a little more advanced than the known program).
      3. If you don't add a goal program, programs that contain adjacent language features to the known program will be shown.
  3. Click Search to see the next programs you can learn.
  4. Next programs you can learn are listed below by program, language features they introduce, and their content; see the following example:

    Program

    This program introduces the language features: A,B,C

    ...program content that uses language features A,B,C...
  5. Below the programs a graph shows the relationship between your Known and Goal programs. The top of the graph being your known program, and the bottom being your goal. The directed edges are followed to determine language features and programs that show up in your search results.
      (hint: If the graph or your search results get too much, try working with programs that are simpler and have fewer differences, and gradually build up to what you want to see.)
